# DistroForge - ChadWM Themes

Repositório de temas ChadWM para integração com o instalador Calamares do projeto DistroForge.

## Temas Inclusos

- chadwm

## Estrutura

```
/
├── themes/           # Arquivos dos temas ChadWM
├── previews/        # Imagens de preview
├── scripts/         # Scripts de instalação
```

## Instalação via Calamares

```bash
sudo ./scripts/[distro]/chadwm.sh
```

## Instalação Manual

```bash
git clone https://github.com/lucasgertke11-bot/theme-chadwm.git
cd theme-chadwm
sudo ./scripts/[distro]/chadwm.sh
```

## Dependências

Os scripts instalam automaticamente:
- dwm, dmenu
- st, picom
- rofi, feh
- nitrogen
- eww (widget)
- Nerd Fonts

## Distros Suportadas

Arch Linux, Ubuntu, Fedora, OpenSUSE, Gentoo
